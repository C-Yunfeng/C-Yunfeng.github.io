二、损失函数及求导

1.表达方式一(求和表达式)：
  • J(θ)J(\theta )J(θ)：

如果仍使用Linear Regression的损失函数的形式：J(θ)=12m∑i=1m(hθ(x(i))−y(i))J(\theta)=\frac{1}{2m}\sum_{i=1}^m(h_\theta(x^{(i)})-y^{(i)})J(θ)=2m1​∑i=1m​(hθ​(x(i))−y(i))，则sigmoid的高度非线性会使J(θ)J(\theta)J(θ)的曲线变为非凸性，导致梯度下降求解困难，因此引入新的损失函数**(为了简化，省略了循环变量iii)**：

J(θ)=−1m∑i[ylog(hθ(x)+(1−y)log(1−hθ(x))]J(\theta)=-\frac{1}{m}\sum_i[ylog(h_\theta(x)+(1-y)log(1-h_\theta(x))] J(θ)=−m1​i∑​[ylog(hθ​(x)+(1−y)log(1−hθ​(x))]

其中，当y=1y=1y=1时，log(hθ(x))log(h_\theta(x))log(hθ​(x))的图像如下：

当y=0y=0y=0时，log(1−hθ(x))log(1-h_\theta(x))log(1−hθ​(x))的图像如下：

  • 对θ\thetaθ求导：

∇θJ(θ)=−1m∑i[(hθ(x)−y)x]\nabla_\theta J(\theta)=-\frac{1}{m}\sum_i[(h_\theta(x)-y)x] ∇θ​J(θ)=−m1​i∑​[(hθ​(x)−y)x]

  • 梯度下降：

θ:=θ−αm∑i[(hθ(x)−y)x]\theta:=\theta-\frac{\alpha}{m}\sum_i[(h_\theta(x)-y)x] θ:=θ−mα​i∑​[(hθ​(x)−y)x]

2.表达方式二(向量表达式)：
  • J(θ)J(\theta)J(θ)

J(θ)=−1m[yTlog(sigmoid(Xθ))+(1−y)Tlog(1−sigmoid(Xθ))]J(\theta)=-\frac{1}{m}[y^Tlog(sigmoid(X\theta))+(1-y)^Tlog(1-sigmoid(X\theta))] J(θ)=−m1​[yTlog(sigmoid(Xθ))+(1−y)Tlog(1−sigmoid(Xθ))]

  • 对θ\thetaθ求导

∇θJ(θ)=−1m[XT(sigmoid(Xθ)−y)]\nabla_\theta J(\theta)=-\frac{1}{m}[X^T(sigmoid(X\theta)-y)] ∇θ​J(θ)=−m1​[XT(sigmoid(Xθ)−y)]

  • 梯度下降：

θ:=θ−αm[XT(sigmoid(Xθ)−y)]\theta:=\theta-\frac{\alpha}{m}[X^T(sigmoid(X\theta)-y)] θ:=θ−mα​[XT(sigmoid(Xθ)−y)]

3.梯度下降代码(Octave)
function [J, grad] = costFunction(theta, X, y)
m = length(y); % number of training examples
J = 1/m*(-(y')*log(sigmoid(X*theta))-(1-y)'*log(1-sigmoid(X*theta))); %计算代价函数
grad = 1/m*X'*(sigmoid(X*theta)-y);  %求梯度,shape as zeros(size(theta))
end
4.注意

值得注意的是，尽管表达式一和二中的θ\thetaθ是一样的，均为n∗1n*1n∗1维的向量；但在表达方式一中，单个样本的假设函数值为θTx(i)\theta^Tx^{(i)}θTx(i)，θ\thetaθ在前；而在表达式二中，全部样本的假设函数值组成的向量为XθX\thetaXθ，θ\thetaθ在后。但本质上是一致的，详细解释↓：

我们平常在表示一个长度为nnn的向量a⃗\vec{a}a时，如果没有明确指明是n∗1n*1n∗1的还是1∗n1*n1∗n的，则默认为n∗1n*1n∗1的，即$$\vec{a}=\left[\begin{matrix}1\2\.\.\n\end{matrix}\right]$$。所以同理，单个样本x(i)=[x1(i)x2(i),,xn(i)]x^{(i)}=\left[\begin{matrix}x_1^{(i)}\\x_2^{(i)}\\,\\,\\x_n^{(i)}\end{matrix}\right]x(i)=⎣⎢⎢⎢⎢⎢⎡​x1(i)​x2(i)​,,xn(i)​​⎦⎥⎥⎥⎥⎥⎤​，总样本矩阵X=[x(1)Tx(2)T,,x(i)T]X=\left[\begin{matrix}x^{(1)T}\\x^{(2)T}\\,\\,\\x^{(i)T}\end{matrix}\right]X=⎣⎢⎢⎢⎢⎡​x(1)Tx(2)T,,x(i)T​⎦⎥⎥⎥⎥⎤​，θ=[θ1θ2,,θn]\theta=\left[\begin{matrix}\theta_1\\\theta_2\\,\\,\\\theta_n\end{matrix}\right]θ=⎣⎢⎢⎢⎢⎡​θ1​θ2​,,θn​​⎦⎥⎥⎥⎥⎤​，因此：

Xθ=[x(1)Tx(2)T..x(i)T]∗θ=[x(1)Tθx(2)Tθ..x(i)Tθ]=[θTx(1)θTx(2)..θTx(i)]X\theta=\left[\begin{matrix}x^{(1)^T}\\x^{(2)T}\\.\\.\\x^{(i)T}\end{matrix}\right]*\theta=\left[\begin{matrix}x^{(1)^T}\theta\\x^{(2)T}\theta\\.\\.\\x^{(i)T}\theta\end{matrix}\right]=\left[\begin{matrix}\theta^Tx^{(1)}\\\theta^Tx^{(2)}\\.\\.\\\theta^Tx^{(i)}\end{matrix}\right] Xθ=⎣⎢⎢⎢⎢⎡​x(1)Tx(2)T..x(i)T​⎦⎥⎥⎥⎥⎤​∗θ=⎣⎢⎢⎢⎢⎡​x(1)Tθx(2)Tθ..x(i)Tθ​⎦⎥⎥⎥⎥⎤​=⎣⎢⎢⎢⎢⎡​θTx(1)θTx(2)..θTx(i)​⎦⎥⎥⎥⎥⎤​

三、加惩罚项的损失函数及求导

1.表达方式一：
  • J(θ)RegJ(\theta )_{Reg}J(θ)Reg​：

J(θ)Reg=J(θ)+Reg=−1m∑i=1m[ylog(hθ(x)+(1−y)log(1−hθ(x))]−λ2m∑j=1nθj2\begin{aligned} J(\theta)_{Reg}&=J(\theta)+Reg\\ &=-\frac{1}{m}\sum_{i=1}^m[ylog(h_\theta(x)+(1-y)log(1-h_\theta(x))]-\frac{\lambda}{2m}\sum_{j=1}^n\theta_j^2\\ \end{aligned} J(θ)Reg​​=J(θ)+Reg=−m1​i=1∑m​[ylog(hθ​(x)+(1−y)log(1−hθ​(x))]−2mλ​j=1∑n​θj2​​

  • 求导

∇θJ(θ)Reg=−1m∑i[(hθ(x)−y)x]−λm∑jθj\nabla_\theta J(\theta)_{Reg}=-\frac{1}{m}\sum_i[(h_\theta(x)-y)x]-\frac{\lambda}{m}\sum_j\theta_j ∇θ​J(θ)Reg​=−m1​i∑​[(hθ​(x)−y)x]−mλ​j∑​θj​

  • 梯度下降

θj:=θj−αm∑i[(hθ(x)−y)x]−λαmθj=θj(1−λαm)−αm∑i[(hθ(x)−y)x]\begin{aligned} \theta_j:&=\theta_j-\frac{\alpha}{m}\sum_i[(h_\theta(x)-y)x]-\frac{\lambda\alpha}{m}\theta_j\\ &=\theta_j(1-\lambda\frac{\alpha}{m})-\frac{\alpha}{m}\sum_i[(h_\theta(x)-y)x] \end{aligned} θj​:​=θj​−mα​i∑​[(hθ​(x)−y)x]−mλα​θj​=θj​(1−λmα​)−mα​i∑​[(hθ​(x)−y)x]​

2.表达方式二：
  • J(θ)RegJ(\theta)_{Reg}J(θ)Reg​：

J(θ)Reg=−1m[yTlog(sigmoid(Xθ))+(1−y)Tlog(1−sigmoid(Xθ))]−λ2mθ2J(\theta)_{Reg}=-\frac{1}{m}[y^Tlog(sigmoid(X\theta))+(1-y)^Tlog(1-sigmoid(X\theta))]-\frac{\lambda}{2m}\theta^2 J(θ)Reg​=−m1​[yTlog(sigmoid(Xθ))+(1−y)Tlog(1−sigmoid(Xθ))]−2mλ​θ2

  • 对θ\thetaθ求导

∇θJ(θ)=−1m[XT(sigmoid(Xθ)−y)]−λEmθ\nabla_\theta J(\theta)=-\frac{1}{m}[X^T(sigmoid(X\theta)-y)]-\frac{\lambda E}{m}\theta ∇θ​J(θ)=−m1​[XT(sigmoid(Xθ)−y)]−mλE​θ

  • 梯度下降：

θ:=θ−αm[XT(sigmoid(Xθ)−y)]−αλEmθ=θ(1−αλEm)−αm[XT(sigmoid(Xθ)−y)]\begin{aligned} \theta:&=\theta-\frac{\alpha}{m}[X^T(sigmoid(X\theta)-y)]-\alpha\frac{\lambda E}{m}\theta\\ &=\theta(1-\alpha\frac{\lambda E}{m})-\frac{\alpha}{m}[X^T(sigmoid(X\theta)-y)] \end{aligned} θ:​=θ−mα​[XT(sigmoid(Xθ)−y)]−αmλE​θ=θ(1−αmλE​)−mα​[XT(sigmoid(Xθ)−y)]​

3.梯度下降代码(Octave)
function [J, grad] = costFunctionReg(theta, X, y, lambda)
m = length(y); % number of training examples
J = 1/m*(-(y')*log(sigmoid(X*theta))-(1-y)'*log(1-sigmoid(X*theta)))+(1/(2*m))*lambda*(sum(theta .^2) - theta(1)^2); %正规化时不用对θ1正规化？

grad = 1/m*X'*(sigmoid(X*theta)-y)+lambda*theta/m;
grad(1) = grad(1)-lambda*theta(1)/m;
end

四、从概率角度解释损失函数

  • 假设有一个二分类问题：0或1；
    我们知道在逻辑回归中，hθ(x)h_\theta(x)hθ​(x) 的本质是概率值，所以我们用概率表示hθh_\thetahθ​的值，且py=0=1−pp_{y=0}=1-ppy=0​=1−p；
    所以第i个样本的概率分布函数可以表示为：

P(y(i)∣x(i))={p,y(i)=11−p,y(i)=0P(y^{(i)}|x^{(i)})=\begin{cases} p & ,y^{(i)}=1 \\ 1-p & ,y^{(i)}=0 \end{cases} P(y(i)∣x(i))={p1−p​,y(i)=1,y(i)=0​

  • 我们可以把上式简化为：P(y(i)∣x(i))=py(i)∗(1−p)1−y(i)P(y^{(i)}|x^{(i)})=p^{y^{(i)}}*(1-p)^{1-y^{(i)}}P(y(i)∣x(i))=py(i)∗(1−p)1−y(i)，注意上标(i)表示属于第i个样本，而pyp^{y}py则表示正常的p的y次方；
  • 所以全体样本的分布（即似然函数）为：P总=∏i=1m[py(i)∗(1−p)1−y(i)]P_{总}=\prod_{i=1}^{m}[p^{y^{(i)}}*(1-p)^{1-y^{(i)}}]P总​=∏i=1m​[py(i)∗(1−p)1−y(i)]
    取对数（因为我们要求代价函数的最值，转化为ln函数方便计算）得：

对数似然函数：ln(P总)=ln(∏i=1m[py(i)∗(1−p)1−y(i)])=∑i=1m[y(i)ln(p)+(1−y(i))ln(1−p)]对数似然函数：ln(P_{总}) = ln(\prod_{i=1}^{m}[p^{y^{(i)}}*(1-p)^{1-y^{(i)}}])=\sum_{i=1}^{m}[y^{(i)}ln(p)+(1-y^{(i)})ln(1-p)] 对数似然函数：ln(P总​)=ln(i=1∏m​[py(i)∗(1−p)1−y(i)])=i=1∑m​[y(i)ln(p)+(1−y(i))ln(1−p)]

  • 令J(θ)=−1mln(P总)J(\theta)=-\frac{1}{m}ln(P_{总})J(θ)=−m1​ln(P总​)，所以最大化似然估计即等价于最小化代价函数。

  • 分类结果可视化(Python)
def visualize_classifier(model, X, y, ax=None, cmap='rainbow'):
    #既可以在ax上画，也可以在figure上画
    ax = ax or plt.gca()
    
    #训练集-散点图
    ax.scatter(X[:, 0], X[:, 1], c=y, s=30, cmap=cmap)
    
    ax.axis('tight')#调整axes之间的间距
    ax.axis('off')#隐藏坐标轴
    
    xlim = ax.get_xlim()
    ylim = ax.get_ylim()
    #对于底盘上每一个点进行分类
    model.fit(X, y)
    xx, yy = np.meshgrid(np.linspace(*xlim, num=200),np.linspace(*ylim, num=200))
    Z = model.predict(np.c_[xx.ravel(), yy.ravel()]).reshape(xx.shape)
    #根据分类画出底色
    n_classes = len(np.unique(y))
    contours = ax.contourf(xx, yy, Z, alpha=0.3,
                           levels=np.arange(n_classes + 1) - 0.5,
                           cmap=cmap)
    #当存在多个不同的数据集要可视化时，设置每个子图为自己的lim
    ax.set(xlim=xlim, ylim=ylim)
import numpy as np 
from sklearn.datasets import make_blobs
from sklearn.linear_model import LogisticRegression
import matplotlib.pyplot as plt

X, y = make_blobs(n_samples=300, centers=2,
                  random_state=0, cluster_std=1.0)
model = LogisticRegression()
visualize_classifier(model, X, y)

2. 数据清洗：

2.1 值缺失：

数值型变量有缺失，可以使用中位数/均值填充，且中位数一般比均值更稳健；类别型变量有缺失，可以使用众数填充；若缺失程度太高，则可以舍弃此变量；

另外，还可以使用KNN类的算法来选择最近似的样本进行填充；

缺失信息有时也可以作为一种特征表达，例如人的性别：男、女、不详都有不同的含义。

  • 查看缺失情况
train.isnull()
train.isnull().sum()
train.isnull().any()
  • 填补
#因为此例中的train里没有NaN，所以把age列值为50~99的置空，实际操作不需要第一步
train.replace({'age':range(50,100)},np.NaN,inplace=True)
train['age'].fillna(train['age'].mean(),inplace=True)
  • 删除
#因此例中train里没有NaN，所以我们先随意把一些值置空，然后再删除
train.replace({'education':'unknown'},np.NaN,inplace=True)
train.dropna(subset=['education'],inplace=True)

3. 区间缩放

3.1 preprocessing.scale

公式：(X-mean)/std ；可以缩放指定列，包括1维和多维；缩放之后，所有维数总均值→0，总方差→1(均值并不总是等于0)；

from sklearn import preprocessing

# train['cons.price.idx']列原来的取值范围为92~95左右
train['cons.price.idx']=preprocessing.scale(train['cons.price.idx'])
# 处理之后每个feature的分布不变(当样本很少时分布变化可能会比较大)
3.2 preprocessing.StandardScaler

公式同上，(X-mean)/std；只能处理多维；处理之后总均值→0，总方差→1；

from sklearn.preprocessing import StandardScaler

data = [[0, 0,0], [0, 0,2], [1, 1,4], [1, 1,0]]
scaler = StandardScaler(copy=False) #参数with_maen和with_std默认为True，即去中心化并将方差缩放为1
data=scaler.fit_transform(data) #copy=False表示在原数据上缩放，但一般没有用，所以需要赋值一下
# 结果：data.mean()=9.25185853854297e-18，data.std()=1.0

4. 特征编码

4.1 独热编码，增维
  • 方法1：pd.get_dummies
# #将所有非数值型变量改为哑变量
train_not_value=X.select_dtypes(include=['object'])
column_names=train_not_value.columns.tolist()

#转为one_hot:
for column_name in column_names:
    dummy_ranks = pd.get_dummies(X[column_name], prefix=column_name)
    cols_to_keep = X.columns.tolist()
    cols_to_keep.remove(column_name)
    X = X[cols_to_keep].join(dummy_ranks.loc[:, column_name:])

#结果：
#marital 转化为四列： marital_divorced, marital_married,marital_single,marital_unknown

4.3 序列编码

有时需要转码之后的数字包含权重信息，如温度的高、中和低。这是一般使用字典序列来编码，这样就保留了高＞中＞低的信息；

# 假设我们有一个DataFrame，其中有一个特征‘温度’，取值为‘high’，‘medium’，‘low’
map_dict={'low':0,'mediun':1,'high':2}
def map_values(x):
    return map_dict[x]
DataFrame['temperature']=DataFrame['temperature'].apply(lambla x:map_values(x))

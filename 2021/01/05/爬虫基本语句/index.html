1.urllib

1.1 request（发送请求）

  • urlopen() 
    urllib.urlopen(url,data=None,timeout=0.5,cafile=None,capath=None,cadefault=False,context=None)

构造了最基本的HTTP请求，同时还带有处理授权验证、重定向、Cookies等功能；

>参数：①.data(需要bytes()将data转化为bytes类型；②.若传递此参数，则请求方式从GET变为POST)；③.cafile和capath分别制定CA证书和其路径，用于请求HTTPS链接；④.cadefault已弃用。

response是一个HTTPResponse类型的对象；
>方法：**read()**，readlines()，getheader(name)，getheaders()，geturl()...
>属性：status，code，version，msg，reason...

from urllib.request import urlopen

response=urlopen('http://chenyunfeng.com/',data=None,timeout=0.1)

# print(response.read().decode('utf-8'))
# response.getcode() #200
# response.info() #<http.client.HTTPMessage at 0x256ab86a470>

  • Request()

    urllib.Request(url,data=None,headers={},origin_req_host=None,unverfiable=False,method=None)

    urlopen()的补充，可以传入更多参数，如Headers

    参数：①.data同urlopen()；②.headers是请求头，类型为dict，常用来修改User-Agent来伪装浏览器，若此处不传，之后也可以通过add_header()来添加；③.origin_req_host指请求方的host名称或IP地址；④.unverifiable表示请求是否是无法验证的；⑤.method指请求使用的方法，如GET、POST和PUT；

    req的类型是urllib.request.Request；我们通过Request()构建一个对象之后仍通过urlopen()来发送请求；

    from urllib.request import Request,urlopen
from urllib import parse

url='http://chenyunfeng.com/'
headers={
    'User-Agent':'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/74.0.3729.169 Safari/537.36',
    'Host':'chenyunfeng.com'
}

req=Request(url=url,headers=headers,method='POST')
response=urlopen(req)
print(response.read().decode('utf-8'))

1.2 error（异常处理）

urllib的error模块定义了由request模块产生的异常，当请求出现异常时，request模块便会抛出error中定义的异常。

  • URLError
    URLError类继承自OSError类，是error异常模块的基类，由request模块产生的异常都可以使用URLError来捕获并处理。 
    from urllib import request,error
try:
    reponse=request.urlopen('http://chenyunfeng.com/')
    print('请求成功')
except error.URLError as e:
    print(e.reason)
  • HTTPError
    是URLError的子类，专门用来处理HTTP请求错误，如网页不存在

    属性：①.code，返回HTTP状态码，如404表示网页不存在，500表示服务器内部错误；②.reason，返回错误原因；③.headers，返回请求头。

    # 我们用一个不存在的网址来尝试
from urllib import request,error
try:
    reponse=request.urlopen('http://chenyunfeng.com/sss')
    print('请求成功')
except error.URLError as e:
    print(e.reason,e.code,e.headers,sep='\n')

1.3 parse

urllib的parse模块定义了处理URL的标准接口，如URL的拆分、合并及转换。

  • urlparse()

    urllib.parse.urlparse(urlstring,scheme='',allow_fragments=True)

    参数：①.urlstring，待解析的URL，标准的格式为scheme://netloc/path;params?query#fragment，即协议://域名/路径;参数?查询条件#锚点；②.scheme，URL的协议，如http、https；③.allow_fragments，是否忽略fragment，若忽略fragment，则它会被解析为path、parameters或query的一部分，而fragment部分为空。

    from urllib.parse import urlparse

result1=urlparse('http://chenyunfeng.com/index.html;user?id=5#title1')
result2=urlparse('http://chenyunfeng.com/index.html;user?id=5#title1',allow_fragments=False)
print("result1:{0}\nresult2:{1}".format(result1,result2))
#↑可以看出，当don't allow fragments时，fragment会被解析为前面属性的一部分，此处为query的一部分
result3=urlparse('http://chenyunfeng.com/index.html#title1',allow_fragments=False)
print('reuslt3:',result3)
#↑可以看出，当params和query为空时，fragment会被解析为path的一部分

  • urlunparse()
    参数长度是一个可迭代的对象，如字典。且属性值可以为空，但长度必须为6，否则会报错。

    from urllib.parse import urlunparse
  
  data=['http','chenyunfeng.com','数据链路层','','','']
  print(urlunparse(data))
